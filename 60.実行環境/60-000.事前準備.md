
**「図解」Strandsエージェント徹底解説** で紹介している各種のStrandsのサンプルコード実行には、以下のような環境が必要です。

- AWS Account
	- 他のモデルプロバイダーでもいいですが、現状ではAnthropic@AWS Bedrockをベースにしています。
- Linux
	- WSL/Mac/EC2
- uv
- venv
- npx

# Python 実行環境

### UVのインストール

uvツールをインストールしてください。

```
curl -LsSf https://astral.sh/uv/install.sh | sh

uv --version
uv 0.9.8
```

uvのPATH設定を実行。ここでは、Unbuntu 22.04(WSL)の例:
```
ls ~/.local/bin/uv
/home/ubuntu/.local/bin/uv

.profile
# set PATH so it includes user's private bin if it exists
if [ -d "$HOME/.local/bin" ] ; then
    PATH="$HOME/.local/bin:$PATH"
fi
```

### 仮想環境作成

既存のvenv環境が存在しても上書きしておいたほうが良いかも。
AI関連ツールの実行環境でuv前提の開設が多いためにやっておいて損はありません。

```
# 既存のvenv環境が存在する場合、上書き

uv venv -p 3.12 .venv

source ./venv/bin/activate
(.venv) ubuntu@LAPTOP-DHL9LRBE:~$
```

仮想環境を永続化。

```
echo "source ./venv/bin/activate" >> ~/.profile
```

```
uv pip install strands-agents strands-agents-tools boto3
```
